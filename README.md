# Themes DaisyUI

A web application for creating, browsing, and sharing custom DaisyUI themes.

## Features

- **Theme Builder**: Create your own DaisyUI themes with a visual editor.
- **Theme Explorer**: Browse and filter themes created by the community.

## Tech Stack

- **Framework**: React / TanStack Start
- **Styling**: Tailwind CSS v4 + DaisyUI v5
- **Animations**: Motion Primitives
- **State Management**: Zustand & TanStack Query
- **Backend/Auth**: Supabase
- **Routing**: TanStack Router
